# 構図ガイドカメラ

スマホ・PCのブラウザだけで動く、構図ガイド付きカメラWebアプリ。プラグイン不要、インストール不要。カメラ映像に構図線（三分割・黄金比など9種）を重ねながら撮影でき、被写体を自動検知して構図が合ったタイミングを光と振動で知らせる。

## 使い方

1. アプリを開き、カメラへのアクセスを許可する。
2. 画面下のガイドストリップから使いたい構図をタップして選ぶ。同じ構図を連続でタップすると、線の配置パターン（バリアント）が切り替わる。
3. 画面上部でアスペクト比・ガイドの濃度・ガイド焼き込み（撮影画像に線を残すか）・おまかせ構図（自動レコメンド）を調整する。
4. 被写体が構図のポイント（スイートスポット）に合うと、ガイドが光って端末が振動する。人物などが検知できない場合は、映像を直接タップして被写体位置を手動指定できる（映像をダブルタップで解除）。
5. 「おまかせ構図」をONにすると、映像内の被写体配置に合いそうな構図が自動で提案され、下部にバッジで表示される。バッジをタップするとその構図に切り替わる。
6. 中央下のシャッターボタンで撮影し、端末に画像を保存する。

### 収録している構図（9種）

- **基本4種**: 三分割法、三角構図、黄金比、白銀比
- **応用5種**: ファイグリッド、レイルマン、対角線構図、シンメトリー、日の丸構図

各構図には複数のバリアント（線の向き・配置パターン）があり、同じ構図アイコンを連続タップすると切り替わる。

## 機能

- **9種の構図ガイド表示**（バリアント切替対応）とアスペクト比・濃度調整
- **被写体自動検知**（TensorFlow.js + COCO-SSD）による人物等の検出
- **構図合致の発光・振動フィードバック** — 被写体が構図のスイートスポットに入ると視覚・触覚で通知
- **手動タップフォールバック** — 検知できない場合、映像タップで被写体位置を指定
- **おまかせ構図レコメンド** — 被写体配置から合う構図をバッジで提案
- **撮影** — 素の映像 or ガイド焼き込み済みの画像として保存

## 技術構成

- Vanilla JavaScript（ESM）、フレームワーク・ビルドツールなし
- 被写体検知に TensorFlow.js / COCO-SSD（CDN読み込み）
- テスト: Node標準テストランナー（`node --test`）

```
src/
  app.js         アプリのエントリ・状態管理
  ui.js          DOM操作・UIイベント
  camera.js      カメラ起動（背面カメラ優先）
  capture.js     撮影・画像保存
  detector.js    COCO-SSDによる被写体検知
  hittest.js     スイートスポットとの合致判定
  recommender.js おまかせ構図レコメンド
  renderer.js    ガイド線の描画
  guides/        構図定義（basic.js / advanced.js）
```

## ローカルで動かす

Python 3が入っていれば、リポジトリ直下で以下を実行する。

```bash
python3 -m http.server 8000
```

ブラウザで `http://localhost:8000` を開く（カメラAPIは `localhost` では制限なしで動作する）。

### テスト

```bash
npm test
```

## スマホでの利用について

カメラAPI（`getUserMedia`）はブラウザの仕様上、**HTTPS**または`localhost`でしか動作しない。スマホの実機で試す場合は、GitHub Pagesなど HTTPS 配信できる場所にデプロイして、そのURLをSafari/Chromeで開く必要がある。ローカルIPアドレス（`http://192.168.x.x:8000`など）ではカメラが起動しないので注意。
